# Analizador de Texto con Estructuras Básicas

Proyecto realizado como práctica académica para analizar texto utilizando:

- Strings
- Lists
- Sets
- Dictionaries
- Manejo de excepciones
- Programación orientada a objetos

## Requisitos para la ejecucion del programa

Python 3.12+

## Cómo ejecutar el programa 

python main.py

## Las modalidades que el programa tiene disponible 

1 - Leer desde archivo (.txt)
2 - Ingresar texto manualmente en consola

## Cómo ejecutar las pruebas

python test_text_analyzer.py
